# Personal_Expense_tracking_API

Это асинхронный REST API для учёта личных доходов и расходов. Проект написан на Python с использованием современного стека: FastAPI, SQLAlchemy (в асинхронном режиме), PostgreSQL и Pydantic версии 2.

Основные возможности:

Регистрация и управление пользователями.
Создание категорий — например, «Еда», «Зарплата», «Транспорт». Каждая категория может быть доходом или расходом (поле is_income).
Добавление транзакций: сумма, описание, дата, привязка к пользователю и категории.
Фильтрация транзакций: по пользователю, категории, типу (доход/расход).
Автоматический расчёт баланса: сумма всех доходов минус сумма всех расходов.
Пагинация (по 10 записей на страницу).
Экспорт данных пользователя в JSON.
Все эндпоинты протестированы и имеют автоматическую документацию — после запуска сервера она доступна по адресу http://localhost:8000/docs (Swagger UI).

Структура проекта разделена по модулям (пользователи, категории, транзакции), используется асинхронная работа с базой данных через asyncpg — это обеспечивает высокую производительность даже при большом числе запросов.

Для запуска нужен PostgreSQL.
